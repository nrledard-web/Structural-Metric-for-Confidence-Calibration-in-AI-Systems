# Structural-Metric-for-Confidence-Calibration-in-AI-Systems
# Mécroyance App

Prototype simple en Python fondé sur la formule :

**M = (G + N) − D**

- **G** : gnosis — savoir articulé
- **N** : nous — intégration et compréhension vécue
- **D** : doxa — certitude ou rigidité assertive

## Objectif

Ce projet explore un modèle qualitatif de calibration cognitive permettant
d’interpréter différents états de cognition selon leur degré de fermeture
ou de révisabilité.
